# 語音時間提示器

這是一個簡單的 Python 程式，使用語音引擎（Text-to-Speech, TTS）來報告目前的時間或星期幾。

## 功能介紹

- 可以語音讀出目前的時間。
- 可以語音讀出今天是星期幾。
- 使用簡單的命令列介面，方便操作。
- 採用內建的語音合成引擎，無需額外安裝其他工具。

## 使用方式

在終端機（命令提示字元）中執行以下指令：


執行後，程式會提示你輸入選項：


請根據需求輸入數字：

- 輸入 `1`：程式會語音播報並在終端顯示目前時間。
- 輸入 `2`：程式會語音播報並在終端顯示今天星期幾。
- 輸入 `3`：程式結束執行。
